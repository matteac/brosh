# Brosh

Brosh is a browser-based virtual shell that simulates a basic Unix-like command-line environment. It allows users to interact with a virtual file system and execute commands directly in their browser.

## Built-In Commands

- **Filesystem Management**:
  - `mkdir`: Create directories
  - `touch`: Create empty files
  - `rm`: Delete files or directories
  - `ls`: List directory contents
  - `cd`: Change current directory
  - `pwd`: Print working directory

- **Utility**:
  - `clear`: Clear the terminal output
  - `add_to_path`: Add directories to the system PATH
  - `print_path`: Display the current PATH
  - `builtins`: List builtin commands

## Getting Started

Open [this](https://matteac.github.io/brosh)

## Usage

Type commands directly into the terminal interface. Use `builtins` to see the full list of available commands.
